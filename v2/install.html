Instalar o aplicativo

Use esta tela para instalar o aplicativo principal em um ambiente limpo e atualizado.

Voltar para o app

Android

Passo a passo

  1. Abra esta pagina no Chrome.
  2. Toque em "Instalar app" se o botao aparecer.
  3. Se nao aparecer, abra o menu do Chrome.
  4. Escolha "Instalar app" ou "Adicionar a tela inicial".

Limpeza

Se ja existe uma instalacao antiga

  1. Desinstale a instalacao antiga da tela inicial.
  2. Limpe os dados do site no Chrome.
  3. Abra o app no navegador.
  4. Instale novamente usando esta pagina.
